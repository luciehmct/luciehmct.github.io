    Benchmarking VLMs for Animal Behavior Analysis with LMMs-Eval

    Campus Biotech (Geneva)

    View repository on GitHub

    Context: This project was my Summer 2025 research internship with the Mathis Group. The goal was to address a key gap in AI: most VLM benchmarks are human-centric and fail to test models on complex ecological tasks. I extended the LMMs-Eval framework to create a standardized, reproducible benchmark for animal behavior analysis using real-world wildlife datasets.

    What I implemented

    • Extended the lmms-eval framework to support new wildlife video datasets (MammAlps and AnimalKingdom).
    • Authored a "universal dataset builder" script to standardize annotations from JSONL files, process videos, and generate HuggingFace-ready datasets.
    • Defined and implemented new evaluation tasks for species, action, and activity recognition.
    • Implemented and registered a Jaccard Index metric to evaluate multi-label predictions and check for partial correctness.
    • Integrated the InternVL3-8B model into the framework for benchmarking.
    • Engineered a robust parsing logic to reliably extract "" tags from model outputs, achieving 100% success on compliant answers and eliminating malformed results seen in baseline tests.
    • Standardized the folder structure for results to automate report generation and simplify debugging.

    My contributions

    My primary contribution was building the end-to-end reproducible pipeline. I was responsible for all dataset processing, task implementation, and metric registration within the lmms-eval framework. I conducted a rigorous comparative analysis between my new pipeline and the original workshop (baseline) results. My analysis confirmed the reproducibility of action recognition, identified key discrepancies in other tasks (likely due to video preprocessing differences), and demonstrated that my pipeline produced cleaner, more reliable parsing than the baseline.
